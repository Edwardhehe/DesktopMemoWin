# 桌面备忘录

一个基于.NET 8 WPF的桌面背景备忘录软件，支持日历形式展示和后台运行。

## 功能特性

- 🖥️ **桌面背景显示**：在桌面右上角显示，仅当用户返回桌面时可见
- 📅 **日历形式展示**：以日历格子形式展示备忘录，支持月份切换
- ✅ **任务管理**：支持标记任务完成，完成后自动排序到最后
- 🎨 **自定义外观**：支持自定义背景颜色和日历样式
- 🔄 **数据迁移**：支持导出/导入数据库文件，实现数据迁移
- 🚀 **开机启动**：支持设置开机自动启动
- 💾 **本地存储**：使用SQLite数据库，数据存储在用户文件夹
- 🔍 **备忘录搜索**：支持快速搜索备忘录内容
- 📋 **备忘录详情**：支持双击查看备忘录详细内容

## 系统要求

- Windows 10 及以上版本
- .NET 8 Runtime

## 技术栈

- **前端框架**：WPF (.NET 8)
- **数据库**：SQLite
- **ORM**：Dapper
- **架构模式**：MVVM

## 构建和运行

### 方法一：使用构建脚本

1. 双击运行 `build.bat`
2. 构建成功后，可执行文件位于 `bin\Release\net8.0-windows\DesktopMemo.exe`

### 方法二：使用命令行
# 还原依赖包
dotnet restore

# 构建项目
dotnet build -c Release

# 运行项目
dotnet run
## 使用说明

### 基本操作

1. **启动应用**：运行 `DesktopMemo.exe`
2. **添加备忘录**：点击日历格子中的 "+" 按钮
3. **标记完成**：点击备忘录前的 "✓" 按钮
4. **删除备忘录**：点击备忘录前的 "✕" 按钮
5. **切换月份**：使用 "◀" 和 "▶" 按钮
6. **查看详情**：双击备忘录项查看详细内容
7. **搜索备忘录**：使用搜索框快速查找备忘录

### 设置功能

点击主窗口右上角的 "⚙" 按钮打开设置窗口：

- **开机启动**：设置是否开机自动启动
- **背景颜色**：自定义日历背景颜色
- **数据管理**：导出/导入/清空备忘录数据

### 数据存储

- 数据库文件位置：`%USERPROFILE%\DesktopMemo\memo.db`
- 支持导出 `.db` 文件到其他位置备份
- 支持从备份文件导入数据

## 开发说明

### 架构设计

项目采用MVVM架构模式：

- **Model**：数据模型层，定义备忘录数据结构
- **View**：视图层，使用XAML定义用户界面
- **ViewModel**：视图模型层，处理业务逻辑和数据绑定

### 核心服务

1. **DatabaseService**：负责SQLite数据库操作
2. **DesktopMonitorService**：监控桌面状态，控制窗口显示/隐藏
3. **StartupService**：管理开机启动设置
4. **SystemTrayService**：提供系统托盘功能，支持最小化到托盘

### 数据绑定

使用WPF数据绑定实现界面与数据的同步：

- 使用 `INotifyPropertyChanged` 接口实现属性变化通知
- 使用值转换器处理数据格式转换
- 使用命令绑定处理用户交互
